IP홀더(IPH) 시나리오

IP를 보유한 크리에이터/에이전시가 플랫폼을 통해 협업 기회를 탐색하는 과정

1 회원가입 및 인증

IPH가 플랫폼에 가입하고 필수 인증을 완료하여 핵심 기능을 사용할 수 있는 상태가 되는 과정

1

플랫폼 접속 및 회원가입 진입

inabooth.io에 접속하여 회원가입 페이지로 이동합니다.

이메일 가입 소셜 가입 (카카오/네이버/구글/LINE)
2

사용자 유형 선택: IP홀더(IPH)

회원가입 시 사용자 유형을 IP홀더로 선택합니다. 가입 후 마이페이지에서 BP로 전환 가능합니다.

IPH-C (개인 크리에이터) IPH-A (에이전시)
3

필수 정보 입력

사용자 유형에 따라 동적으로 필수 입력 필드가 노출됩니다.

이메일/비밀번호 작가 이름/에이전시명 국적 사업자 등록 여부 약관 동의
4

휴대폰 인증 (핵심 기능 사용 시)

IP 등록, 프로젝트 지원, 문의 확인 등 핵심 기능 최초 사용 시 휴대폰 인증이 요구됩니다.

인증번호 발송 코드 입력 인증 완료
5

사업자 인증 (IPH-A 전용)

에이전시(IPH-A)는 사업자등록증 업로드 후 OCR 자동 추출 → 국세청 API 검증을 통해 인증합니다.

가입 및 인증 완료

이제 IP 등록, 오픈 프로젝트 지원 등 모든 IPH 기능을 사용할 수 있습니다.

2 캐릭터(IP) 신규 등록

IPH가 보유한 캐릭터 정보를 플랫폼에 등록하여 BP의 탐색 및 협업 제안 대상이 되는 과정

1

캐릭터 등록 진입

상단 네비게이션 '캐릭터 등록' 버튼 또는 마이페이지에서 등록 시작. 최초 진입 시 튜토리얼이 노출됩니다.

휴대폰 인증 확인 튜토리얼 확인/스킵
2

기본 정보 입력

캐릭터의 기본 속성 정보를 입력합니다. 입력 10초 이상 중단 시 자동 임시저장됩니다.

기본 정보 입력 항목
1 캐릭터 이름
→
2 대표 이미지 + AI 배경색 제안
→
3 프레임 선택
→
4 원작 국가
→
5 카테고리
→
6 태그
3

상세 정보 입력

포트폴리오, 채널 인증, 저작권 정보 등 협업 판단에 필요한 상세 정보를 입력합니다.

포트폴리오 작성 (200자 이상, 이미지 1장 이상) 대표 채널 인증 (API 연동) 저작권 등록번호 (선택) 스타일 가이드북 업로드 (선택)
4

협업 정보 입력

협업 가능 범위와 희망 조건을 설정합니다.

IP 활용 가능 범위 (필수, 복수 선택) 최소 협업 금액(MG) 설정 (선택) 협업 이력 추가 (선택)
5

공개 범위 설정 및 최종 등록

공개/비공개 설정 후 최종 등록합니다. 운영팀 승인 후 탐색 결과에 노출됩니다.

IP 등록 완료

등록된 IP는 BP 탐색 결과에 노출되며, IP 정보품질 지수와 IPH 활동지수에 반영됩니다.

3 오픈 프로젝트 탐색 및 지원

IPH가 BP가 등록한 오픈 프로젝트를 탐색하고 자신의 IP로 지원하는 과정

1

오픈 프로젝트 메뉴 진입

상단 네비게이션 '오픈 프로젝트' 클릭 → 프로젝트 목록 페이지로 이동합니다.

맞춤 콜라보 마감 임박 콜라보 인기 콜라보 전체 콜라보
2

프로젝트 필터링 및 검색

협업 유형, 카테고리, 예산 범위 등 다양한 기준으로 원하는 프로젝트를 탐색합니다.

3

프로젝트 상세 정보 확인

프로젝트 카드 클릭 → 상세페이지에서 협업 조건, 모집 분야, 참여 혜택, 유의사항 등을 확인합니다.

4

프로젝트 지원하기

'지원하기' 버튼 클릭 → 지원 자격 검증 → 지원할 IP 선택 → 양식 작성 → 최종 제출

지원 프로세스
1 자격 검증
→
2 IP 선택
→
3 양식 작성
→
4 가이드북 첨부
→
5 최종 제출
5

지원 현황 관리

마이페이지에서 지원한 프로젝트 목록과 상태를 확인합니다.

선정 시

BP로부터 선정 알림 → 1:1 채팅방 생성 → 협업 조건 협의 → 계약 체결

미선정 시

미선정 알림 수신 → 다른 프로젝트 탐색 및 지원 계속

4 BP 문의 응답 및 협업 진행

BP로부터 협업 문의를 받고 1:1 채팅을 통해 협업을 진행하는 과정 (문의 기반 프로젝트)

1

문의 알림 수신

BP가 IP 상세페이지에서 '문의하기'를 통해 문의를 보내면 알림을 통해 확인합니다.

2

1:1 채팅방에서 문의 내용 확인

자동 생성된 채팅방에서 BP의 문의 내용을 확인합니다.

3

협업 조건 협의

채팅을 통해 세부 조건을 협의하고 파일/자료를 공유합니다.

실시간 메시지 송수신 파일 첨부 읽음 확인
4

제안 수락/거절

BP의 최종 제안을 검토하고 수락 또는 거절합니다.

상태: 문의 중 → 협의 중 → 결제 대기
5

계약 체결 및 협업 진행

BP 결제 완료 후 계약이 체결되며, 프로젝트 상태가 '협업 진행중'으로 변경됩니다.

6

프로젝트 완료 및 상호 평가

협업 완료 후 '완료' 상태로 변경 → 상호 평가 진행

협업 완료

평가 결과는 협업지수에 반영되어 향후 매칭 우선순위에 긍정적 영향을 미칩니다.

브랜드 파트너(BP) 시나리오

IP를 활용하여 제품 생산, 프로모션 등 협업을 진행하려는 기업의 사용 흐름

1 회원가입 및 사업자 인증

BP가 플랫폼에 가입하고 사업자 인증을 완료하여 IP 문의 및 프로젝트 등록이 가능한 상태가 되는 과정

1

회원가입 및 유형 선택

이메일 또는 소셜 로그인으로 가입 후 사용자 유형을 '브랜드 파트너(BP)'로 선택합니다.

2

필수 정보 입력

BP 유형에 맞는 필수 정보를 입력합니다.

회사 이름 담당자 이름 사업자 유형 회사 국적 회사 업종
3

선택 정보 입력

주요 제품/서비스, 선호 IP 스타일을 입력하면 맞춤 추천 정확도가 향상됩니다.

4

휴대폰 인증 + 사업자 인증

문의하기, 프로젝트 등록 등 핵심 기능 사용 시 휴대폰 인증 및 사업자 인증이 필수입니다.

사업자등록증 업로드 OCR 자동 추출 국세청 API 검증

가입 및 인증 완료

이제 IP 탐색/문의, 오픈 프로젝트 등록 등 모든 BP 기능을 사용할 수 있습니다.

2 IP 탐색 및 문의

BP가 협업하고 싶은 IP를 탐색하고 IPH에게 직접 문의하여 협업을 시작하는 과정

1

IP 탐색 메뉴 진입

상단 네비게이션 '탐색' 메뉴 또는 메인 검색창을 통해 IP 탐색을 시작합니다.

2

IP 검색 및 필터링

키워드 검색, 카테고리 필터, 정렬 기능을 활용하여 원하는 IP를 탐색합니다.

기본 필터: IP 유형, 원작 국가, 타깃 팬층 상세 필터 (유료회원): 최소 협업 금액, 프로젝트 이력
3

IP 상세 정보 확인

IP 카드 클릭 → 상세페이지에서 포트폴리오, 협업 이력, 스타일 가이드북 등을 확인합니다.

포트폴리오 협업 이력 MG 정보 스타일 가이드북
4

문의하기

'문의하기' 버튼 클릭 → 사전 조건 검증 → 문의 폼 작성 → 제출

문의 폼 필수 항목
Q1 협업 유형
Q2 예상 예산
Q3 희망 일정
Q4 추가 요청사항

1:1 채팅방 자동 생성

문의 제출 시 IPH와의 1:1 채팅방이 자동 생성되어 협의를 진행할 수 있습니다.

3 오픈 프로젝트 등록

BP가 공개 모집 프로젝트를 등록하여 다수의 IPH로부터 지원을 받는 과정

1

오픈 프로젝트 등록 시작

'오픈 프로젝트' 메뉴에서 '프로젝트 등록' 클릭. 휴대폰 인증 및 사업자 인증이 필수입니다.

2

기본 정보 입력

프로젝트의 기본 정보를 입력합니다.

프로젝트 제목 대표 이미지 주요 협업 유형 모집 마감일 희망 IP 카테고리 프로젝트 예산
3

상세 정보 입력

모집 분야, 디자인 방향, 참여 혜택, 유의사항 등 상세 정보를 작성합니다.

인사말 및 프로젝트 개요 (200자 이상) 모집 분야 (50자 이상) 참여 IPH 혜택 (50자 이상) 유의사항 (50자 이상)
4

등록 완료 및 승인 대기

프로젝트 등록 후 운영팀 검토를 거쳐 승인되면 공개됩니다.

5

지원자 관리 및 IP 선정

IPH 지원 현황을 확인하고, 지원서를 검토하여 적합한 IP를 선정합니다.

IP 선정 완료

선정된 IPH와 1:1 채팅방이 생성되어 세부 조건 협의 → 결제 → 계약 체결로 진행됩니다.

4 협업 진행 및 완료

BP가 IPH와 협업 조건을 협의하고 결제/계약 체결 후 프로젝트를 완료하는 과정

1

채팅방에서 협업 조건 협의

1:1 채팅방에서 세부 조건을 협의합니다.

상태: 문의 중 → 협의 중
2

제안 및 수락

최종 제안을 보내고 IPH의 수락을 기다립니다. 수락 시 '결제 대기' 상태로 전환됩니다.

3

결제 및 계약 체결

결제 대기 상태에서 결제 모달이 노출되고, 결제 완료 시 계약이 자동 체결됩니다.

상태: 결제 대기 → 계약 체결 → 협업 진행중
4

프로젝트 완료 및 상호 평가

협업 완료 후 '완료' 상태로 변경하고, 상호 평가를 진행합니다.

협업 완료

평가 결과는 양측의 협업지수에 반영됩니다.

공통 시나리오

IPH와 BP 모두 사용하는 공통 기능의 사용 흐름

1 마이페이지 및 계정 관리

모든 사용자가 프로필, 계정, 요금제, 알림 설정 및 플랫폼 내 주요 활동을 관리하는 과정

1

프로필 관리

프로필 이미지, 닉네임/회사명, 소개글 등을 수정합니다.

2

계정/인증 관리

이메일 변경, 휴대폰 재인증, 사업자 인증, 소셜 계정 연동 등을 관리합니다.

이메일 변경 휴대폰 인증 사업자 인증 소셜 계정 연동 비밀번호 변경
3

요금제 및 결제 관리

현재 구독 요금제 확인, 업그레이드/다운그레이드, 결제 수단 관리를 수행합니다.

일반 (Basic) - 무료 프로 (Pro) - 유료, 단일 계정 비즈니스 (Business) - 유료, 복수 계정
4

알림 설정

이메일/카카오톡/SMS 알림 수신 여부를 유형별로 설정합니다.

5

사용자 유형 전환

IPH ↔ BP 간 자유롭게 전환 가능합니다. 전환 시 해당 유형의 필수 입력 항목을 충족해야 합니다.

2 인사이트 콘텐츠 탐색

캐릭터/콜라보 관련 시장 트렌드, 뉴스 등 업계 인사이트 콘텐츠를 탐색하는 과정

1

인사이트 메뉴 진입

상단 네비게이션 '인사이트' 클릭. 비회원 포함 모든 사용자 접근 가능합니다.

2

콘텐츠 탐색

'최신 인사이트'와 '맞춤 인사이트' 섹션에서 관심 콘텐츠를 탐색합니다.

카테고리별 필터 기간별 필터 인기순 정렬
3

상세 조회 및 스크랩

콘텐츠 카드 클릭 → 상세 페이지에서 전문 확인 → 스크랩/공유

3 스크랩 및 공유

관심 있는 IP, 프로젝트, 인사이트를 저장하거나 외부에 공유하는 과정

1

스크랩하기

IP 카드, 프로젝트 카드, 인사이트 카드에서 스크랩 버튼 클릭. 비회원은 로그인 유도 모달이 노출됩니다.

2

스크랩 관리

마이페이지에서 스크랩한 IP, 프로젝트, 인사이트를 통합 관리합니다.

3

공유하기

공유 버튼 클릭 → URL 복사 또는 SNS 공유

프로젝트 유형

  • 문의 기반 프로젝트: BP가 IP 탐색 → 직접 문의 → 협의 → 계약 체결
  • 공개 모집 프로젝트: BP가 오픈 프로젝트 공고 → IPH 지원 → 선정 → 계약 체결

협업 상태 흐름

  • 문의 중 → 협의 중 → 결제 대기 → 계약 체결
  • 계약 체결 → 협업 진행중 → 프로젝트 완료
  • 완료 후 상호 평가 → 협업지수 반영

인증 체계

  • 휴대폰 인증: 핵심 기능 사용 시 필수
  • 사업자 인증: IPH-A, BP 필수 (OCR + 국세청 API)

회원 등급별 기능

  • 일반(Basic): 기본 기능, IP 1개 등록 제한
  • 프로(Pro): IP 무제한 등록, 상세 필터/지수 조회
  • 비즈니스(Business): 프로 기능 + 복수 계정, 관리자 초대
