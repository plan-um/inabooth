자주 묻는 질문

INABOOTH 이용에 대한 궁금증을 해결해 드립니다

계정 및 인증

휴대폰 인증이 진행되지 않는 경우, 아래 사항을 확인해 주세요:

1. 입력하신 휴대폰 번호가 정확한지 확인해 주세요.
2. 스팸 문자함을 확인해 주세요. 인증 번호가 스팸으로 분류되었을 수 있습니다.
3. 통신사 문자 수신 차단 설정이 되어있는지 확인해 주세요.
4. 잠시 후 다시 시도해 주세요. 일시적인 통신 장애일 수 있습니다.

문제가 지속되면 고객센터로 문의해 주세요.

도움이 되었나요?

사업자 인증은 사용자 유형에 따라 다릅니다:

IPH-C (개인 크리에이터): 선택 사항입니다. 다만, 사업자 인증을 완료하면 '인증된 IP' 배지가 부여되어 브랜드의 신뢰도가 높아집니다.

IPH-A (에이전시): 필수입니다. 에이전시로 활동하려면 사업자등록증 인증이 필요합니다.

BP (브랜드 파트너): 필수입니다. 브랜드로서 IP와 협업하려면 사업자 인증이 필요합니다.

도움이 되었나요?

비밀번호를 잊으셨다면 아래 단계를 따라 재설정하세요:

1. 로그인 페이지에서 '비밀번호 찾기'를 클릭합니다.
2. 가입 시 사용한 이메일 주소를 입력합니다.
3. 이메일로 전송된 비밀번호 재설정 링크를 클릭합니다.
4. 새 비밀번호를 설정합니다.

재설정 링크는 24시간 동안 유효합니다. 이메일이 도착하지 않으면 스팸함을 확인하거나 다시 요청해 주세요.

도움이 되었나요?

이메일 변경은 마이페이지에서 가능합니다:

1. 마이페이지 > 프로필 관리로 이동합니다.
2. 이메일 항목의 '변경' 버튼을 클릭합니다.
3. 새 이메일 주소를 입력하고 인증을 진행합니다.
4. 새 이메일로 발송된 인증 링크를 클릭하면 변경이 완료됩니다.

보안을 위해 비밀번호 확인이 필요할 수 있습니다.

도움이 되었나요?

IP 등록 및 관리

IP 등록은 4단계로 진행됩니다:

STEP 1 - 기본 정보: 캐릭터 이름, 대표 이미지, 카테고리 등 기본 정보를 입력합니다.
STEP 2 - 상세 정보: 포트폴리오, 대표 채널, 저작권 등록번호 등을 입력합니다.
STEP 3 - 협업 정보: 활용 가능 범위, 최소 협업 금액, 협업 이력 등을 설정합니다.
STEP 4 - 공개 설정: 공개/비공개를 선택하고 등록을 완료합니다.

등록 후 운영팀의 검수를 거쳐 승인되면 탐색 페이지에 노출됩니다.

도움이 되었나요?

네, 등록한 IP는 언제든 수정할 수 있습니다:

1. 마이부스 > IP 관리에서 수정할 IP를 선택합니다.
2. '수정' 버튼을 클릭합니다.
3. 원하는 정보를 수정한 후 저장합니다.

단, 일부 중요 정보(저작권 등록번호 등) 변경 시 운영팀의 재검수가 필요할 수 있습니다.

도움이 되었나요?

저작권 등록번호를 인증하면 다음과 같은 혜택이 있습니다:

1. '법적검증' 배지가 IP에 표시되어 브랜드 신뢰도가 높아집니다.
2. BP(브랜드 파트너)의 상세 필터에서 우선 노출됩니다.
3. 협업 제안을 받을 확률이 높아집니다.
4. 향후 분쟁 발생 시 권리 보호가 용이합니다.

도움이 되었나요?

프로젝트 지원

오픈 프로젝트 지원은 다음 단계로 진행됩니다:

1. 오픈 프로젝트 페이지에서 관심 있는 프로젝트를 선택합니다.
2. '지원하기' 버튼을 클릭합니다.
3. 지원할 IP를 선택합니다 (공개된 IP만 선택 가능).
4. BP가 요청한 양식에 맞게 정보를 입력합니다.
5. 지원서를 제출합니다.

지원 후 BP의 검토를 거쳐 선정/미선정 결과를 받게 됩니다.

도움이 되었나요?

네, 특정 조건에서 지원 취소가 가능합니다:

- 프로젝트가 아직 '모집중' 상태일 때만 취소 가능합니다.
- 마이부스 > 지원 현황에서 해당 프로젝트를 선택합니다.
- '지원 취소' 버튼을 클릭합니다.

주의: 지원 취소 후 같은 프로젝트에 재지원할 경우, 새로운 지원서로 처리됩니다.

도움이 되었나요?

선정 결과는 다음 방법으로 확인할 수 있습니다:

1. 알림: 선정/미선정 시 이메일과 앱 내 알림이 발송됩니다.
2. 마이부스: 마이부스 > 지원 현황에서 지원 상태를 확인할 수 있습니다.
3. 채팅: 선정 시 BP와의 1:1 채팅방이 자동 생성됩니다.

알림을 받지 못하셨다면 알림 설정을 확인해 주세요.

도움이 되었나요?

결제 및 정산

INABOOTH는 사용자 유형별로 다양한 요금제를 제공합니다:

IPH (IP 홀더):
- Basic (무료): IP 1개 등록, 월 1회 프로젝트 지원
- Pro: IP 무제한 등록, 지원 횟수 무제한, 상세 지표 조회
- Business: Pro 기능 + 복수 계정, 관리자 초대

BP (브랜드 파트너):
- Basic (무료): 월 2건 문의, 프로젝트 1개 등록
- Pro: 무제한 문의/등록, 상세 필터/정렬, IP 협업지수 조회
- Business: Pro 기능 + 복수 계정, 관리자 초대

도움이 되었나요?

요금제 업그레이드는 간단합니다:

1. 마이페이지 > 요금제 관리로 이동합니다.
2. 원하는 요금제를 선택합니다.
3. 결제 정보를 입력하고 결제를 완료합니다.

업그레이드 시 남은 기간은 일할 계산되어 새 요금제 기간에 포함됩니다.

도움이 되었나요?

협업 완료 후 정산 절차는 다음과 같습니다:

1. 협업이 완료되면 BP가 결제한 금액이 정산 대기 상태가 됩니다.
2. 마이페이지 > 계좌 관리에서 출금 계좌를 등록합니다.
3. '출금 신청' 버튼을 클릭합니다.
4. 신청 후 영업일 기준 3~5일 내에 등록된 계좌로 입금됩니다.

정산 수수료는 협업 금액의 10%입니다.

모바일 앱은 2025년 상반기 출시 예정입니다. 뉴스레터를 구독하시면 앱 출시 소식을 가장 먼저 받아보실 수 있습니다.

도움이 되었나요?

INABOOTH는 다음 브라우저를 지원합니다:

- Chrome (최신 버전)
- Safari (최신 버전)
- Firefox (최신 버전)
- Edge (최신 버전)

Internet Explorer는 지원하지 않습니다. 최상의 경험을 위해 Chrome 또는 Safari 사용을 권장합니다.

도움이 되었나요?

INABOOTH는 사용자의 개인정보 보호를 최우선으로 합니다:

- 모든 데이터는 SSL/TLS 암호화로 전송됩니다.
- 개인정보는 국내 보안 인증을 받은 서버에서 관리됩니다.
- 개인정보보호법 및 정보통신망법을 준수합니다.
- 제3자에게 동의 없이 정보를 제공하지 않습니다.

자세한 내용은 개인정보처리방침을 확인해 주세요.
