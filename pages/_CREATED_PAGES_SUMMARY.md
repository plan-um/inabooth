## Key Design Features

### CSS Variables (Design System)
- **Colors**: Primary (#4F46E5 Indigo), Secondary (#EC4899 Pink), Success, Warning, Error
- **Typography**: Pretendard Variable font, 9 size scales (xs to 5xl)
- **Spacing**: 10-step spacing scale (0.25rem to 6rem)
- **Radius**: 5 border radius options (6px to full circle)
- **Shadows**: Card, MD, LG shadows with consistent depth

### Component Styles
- **Buttons**: Primary, Secondary, Outline variants with hover states
- **Page Sections**: Max-width 900px, white cards with shadow
- **Steppers**: Circle indicators with active/completed states (on flow pages)
- **Radio Cards**: Card-style selection with hover and selected states
- **Footer**: Dark background with 4-column grid layout

### Special Pages
- **8-1 프로젝트 지원.html**: Full stepper, info box with requirements
- **8-1-1**: Verification status cards with icons
- **8-1-2**: IP selection with card-style radio buttons (NO dropdowns)
