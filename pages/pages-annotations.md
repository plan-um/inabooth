# INABOOTH 페이지 주석

개발자용 스토리보드 주석 문서

---

## 1. Auth (회원가입/로그인)

---

**1-1-1 회원가입**

**1. 이메일 입력**
가입용 이메일 주소 입력. 중복 체크는 서버에서 처리.

**2. 비밀번호 입력**
8자 이상 필수. blur 시 클라이언트 유효성 검사 실행. 미충족 시 에러 메시지 표시 및 input 테두리 빨간색 변경.

**3. 비밀번호 확인**
비밀번호 필드와 일치 여부 검증. 불일치 시 에러 메시지 표시.

**4. 전체 동의 체크박스**
클릭 시 하위 3개 체크박스(이용약관, 개인정보 수집, 마케팅 정보 수신) 모두 체크/해제.

**5. 약관 체크박스**
이용약관, 개인정보 수집 동의는 필수(required). 마케팅 정보 수신은 선택.

**6. 다음 버튼**
필수 약관 미동의 또는 비밀번호 유효성 검사 실패 시 alert 표시. 성공 시 [1-1-2-회원가입_유형-선택]으로 이동.

**7. 소셜 로그인 버튼**
카카오, 네이버, 구글, Apple 4종 제공. 클릭 시 OAuth 플로우 진행 후 [1-1-2-회원가입_유형-선택]으로 이동.

**8. 로그인 링크**
하단 "이미 계정이 있으신가요?" 클릭 시 [1-2-1-로그인]으로 이동.

---

**1-1-2 회원가입_유형 선택**

**1. 사용자 유형 카드**
3가지 유형 중 하나 선택:
- IPH-C (개인 크리에이터): 개인 IP 홀더
- IPH-A (에이전시): 여러 IP 관리 에이전시
- BP (브랜드 파트너): IP와 협업하려는 브랜드/기업

**2. 유형 선택 시 UI 변화**
선택된 카드는 배경색 primary-light, 테두리 primary, 아이콘 배경 primary로 변경. 우측 체크 아이콘 활성화.

**3. 다음 버튼**
유형 미선택 시 disabled 상태. 선택 후 활성화되며 각 유형에 맞는 정보입력 페이지로 이동:
- IPH-C → [1-1-3-1-회원가입_IPH-C-정보입력]
- IPH-A → [1-1-3-2-회원가입_IPH-A-정보입력]
- BP → [1-1-3-3-회원가입_BP-정보입력]

**4. 이전 버튼**
history.back() 호출하여 이전 페이지로 이동.

---

**1-1-3-1 회원가입_IPH-C 정보입력**

**1. 상단 배지**
선택된 유형(IPH-C 개인 크리에이터) 표시. 다른 유형 페이지에서는 해당 유형으로 변경.

**2. 작가 이름**
활동명 또는 본명 입력. 필수 항목.

**3. 국적 선택**
셀렉트박스로 국가 선택. 대한민국, 미국, 일본 등 주요 국가 + 기타 옵션.

**4. 사업자 등록 여부**
개인/사업자 2가지 중 선택. 카드 형태 라디오 버튼.
- 개인: 사업자 미등록 개인 크리에이터
- 사업자: 사업자등록증 보유 (향후 사업자 인증 필요)

**5. 가입 완료 버튼**
모든 필수 항목 입력 후 [1-1-4-회원가입_완료]로 이동.

---

**1-1-3-2 회원가입_IPH-A 정보입력**

**1. 에이전시명**
에이전시 상호명 입력. 필수.

**2. 대표자명**
에이전시 대표자 이름 입력. 필수.

**3. 사업자등록번호**
10자리 사업자등록번호 입력. 형식: 000-00-00000. 필수.

**4. 회사 규모**
셀렉트박스로 선택: 1-10명, 11-50명, 51-100명, 100명 이상.

**5. 가입 완료 버튼**
에이전시는 사업자 인증 필수이므로 가입 후 [9-2-2-사업자-인증] 안내.

---

**1-1-3-3 회원가입_BP 정보입력**

**1. 회사/브랜드명**
기업 또는 브랜드 이름 입력. 필수.

**2. 담당자명**
협업 담당자 이름 입력. 필수.

**3. 담당자 연락처**
연락 가능한 전화번호 입력. 필수.

**4. 사업자등록번호**
10자리 사업자등록번호 입력. BP는 사업자 인증 필수.

**5. 가입 완료 버튼**
[1-1-4-회원가입_완료]로 이동. 이후 사업자 인증 필요 안내.

---

**1-1-3-4 회원가입_BP 선택정보**

**1. 관심 IP 유형**
복수 선택 가능한 체크박스. 캐릭터, 일러스트, 웹툰, 애니메이션 등.

**2. 예상 협업 규모**
셀렉트박스: 소규모(100만원 미만), 중규모(100-500만원), 대규모(500만원 이상).

**3. 관심 협업 형태**
라이선싱, 콜라보 상품, 이벤트/팝업 등 복수 선택.

**4. 건너뛰기 버튼**
선택정보 미입력 시에도 가입 완료 가능.

---

**1-1-4 회원가입_완료**

**1. 성공 아이콘**
체크 아이콘이 scale 애니메이션과 함께 표시.

**2. 다음 단계 안내**
IP 탐색, 오픈 프로젝트 확인, 인사이트 조회 등 시작 가이드 표시.

**3. 시작하기 버튼**
클릭 시 [index](메인 페이지)로 이동. 로그인 상태로 전환.

---

**1-2-1 로그인**

**1. 이메일/비밀번호 입력**
가입된 계정 정보 입력.

**2. 로그인 상태 유지**
체크 시 세션 만료 기간 연장 (remember me 토큰 발급).

**3. 비밀번호 찾기**
[1-2-2-로그인_비밀번호-찾기]로 이동.

**4. 로그인 버튼**
인증 성공 시 [index]로 이동. 실패 시 에러 메시지 표시.

**5. 소셜 로그인**
카카오, 네이버, 구글, Apple OAuth 로그인. 기존 연동 계정 있으면 바로 로그인, 없으면 연동 후 로그인.

**6. 회원가입 링크**
[1-1-1-회원가입]으로 이동.

---

**1-2-2 로그인_비밀번호 찾기**

**1. 가입 이메일 입력**
가입 시 사용한 이메일 주소 입력.

**2. 안내 박스**
이메일 발송 안내, 24시간 유효, 스팸 확인 요청 등 표시.

**3. 재설정 링크 발송 버튼**
클릭 시 성공 메시지 표시, 폼 숨김 처리. 3초 후 [1-2-1-로그인]으로 자동 이동.

**4. 로그인으로 버튼**
[1-2-1-로그인]으로 이동.

---

**1-2-3 로그인_비밀번호 재설정**

**1. 이메일 링크 인증**
이메일로 받은 링크 클릭 시 토큰 검증 후 이 페이지 표시.

**2. 새 비밀번호 입력**
8자 이상 필수. 클라이언트 유효성 검사.

**3. 새 비밀번호 확인**
일치 여부 검증.

**4. 비밀번호 변경 버튼**
변경 성공 시 [1-2-1-로그인]으로 이동하며 성공 메시지 표시.

---

**1-3-1 휴대폰 인증**

**1. 인증 필요 시점 안내**
IP 등록, 프로젝트 지원, 문의하기 시 필요함 표시.

**2. 휴대폰 번호 입력**
11자리 숫자만 입력 가능. 자동 하이픈 없음.

**3. 인증번호 발송 버튼**
번호 10자리 미만 시 alert. 성공 시 step2 표시, 번호 입력란 disabled 처리.

**4. 타이머**
3분(180초) 카운트다운. 0:00 도달 시 "인증번호 만료" 알림.

**5. 인증번호 입력**
6자리 숫자 입력.

**6. 재발송 버튼**
타이머 리셋, 새 인증번호 발송.

**7. 인증 완료 버튼**
6자리 미입력 또는 만료 시 실패. 성공 시 모달 닫힘 또는 이전 페이지로 복귀.

---

**1-3-2 사업자 인증**

**1. 인증 대상 안내**
IPH-A 에이전시, IPH-C 사업자, BP 브랜드 파트너 대상.

**2. 파일 업로드 영역**
클릭 또는 드래그&드롭으로 파일 선택. JPG, PNG, PDF 지원, 최대 10MB.

**3. OCR 자동 인식**
파일 업로드 후 1.5초 딜레이 후 OCR 결과 표시. 사업자등록번호, 상호명, 대표자명, 사업장 주소 자동 추출.

**4. OCR 결과 프리뷰**
추출된 정보 표시. 오류 시 수동 수정 가능.

**5. 인증하기 버튼**
파일 업로드 전 disabled. 업로드 후 활성화. 클릭 시 국세청 API로 사업자 진위 확인 후 인증 완료 처리.

## 2. Main (메인)

---

**2-1 메인 (index.html)**

**1. 히어로 섹션**
검색창과 인기 태그 표시. 태그 클릭 시 검색창에 키워드 자동 입력.

**2. 인기 태그**
최근 인기 검색어 표시. 클릭 시 검색창에 해당 키워드 입력.

**3. IP 탐색 드롭다운**
카테고리별 IP 탐색 메뉴. 캐릭터, 일러스트, 브랜드, 예술 카테고리 제공.

**4. 맞춤 추천 섹션**
로그인 사용자의 관심사/활동 기반 추천 IP 목록. 비로그인 시 인기 IP 표시. 스크랩/조회 데이터 기반 추천 알고리즘 적용.

**5. 급상승 섹션**
최근 조회수/스크랩 급증 IP 표시. '인기' 배지 표시. 주간 단위 데이터 집계.

**6. 최신 등록 섹션**
신규 등록 IP 최신순 표시. 'NEW' 배지 표시.

**7. IP 카드**
썸네일, IP명, 크리에이터명, 카테고리 태그 표시. 클릭 시 [3-2-IP-상세]로 이동. 스크랩 버튼은 로그인 필요.

**8. IP 카드 스크랩 버튼**
클릭 시 스크랩 토글. 비로그인 시 로그인 유도. 스크랩 카운트 증가.

**9. 성공 사례 섹션**
브랜드-IP 협업 사례 캐러셀. 자동 슬라이드 또는 좌우 버튼으로 탐색.

**10. 사용자 유형별 안내**
IPH(IP 홀더)와 BP(브랜드 파트너) 각각의 서비스 이용 흐름 설명.

**11. FAQ 섹션**
아코디언 형태의 자주 묻는 질문. 클릭 시 답변 펼침/접힘. 한 번에 하나만 열림.

**12. 요금제 섹션**
Basic(무료), Pro, Business 3가지 요금제 카드 표시. IP홀더/브랜드파트너 탭으로 전환. 자세히 보기 클릭 시 [9-4-1-요금제_IPH] 또는 [9-4-2-요금제_BP]로 이동.

**13. 뉴스레터 구독**
이메일 입력 후 구독 신청. 최신 IP 소식, 협업 기회 안내 메일 수신.

## 3. Browse (탐색)

---

**3-1 탐색**

**1. 검색창**
키워드 검색. Enter 또는 검색 버튼 클릭 시 결과 필터링.

**2. 카테고리 필터**
캐릭터, 일러스트, 브랜드, 예술 등 대분류 필터. 복수 선택 가능.

**3. 상세 필터 버튼**
클릭 시 [3-1-2-탐색_상세-필터] 모달 또는 패널 표시.

**4. 정렬 드롭다운**
최신순, 인기순, 조회순, 스크랩순 등 정렬 옵션. Pro 전용 정렬(신뢰지수순 등)은 잠금 표시.

**5. IP 그리드**
필터/검색 결과에 따른 IP 카드 목록. 무한 스크롤 또는 페이지네이션.

**6. 결과 없음 상태**
검색 결과 없을 시 안내 메시지 및 필터 초기화 버튼 표시.

---

**3-1-1 탐색_기본 필터**

**1. 카테고리 칩**
캐릭터, 일러스트, 브랜드, 예술 등 선택/해제 가능한 칩 버튼.

**2. 스타일 칩**
귀여운, 힐링, 트렌디, 레트로 등 스타일 필터.

**3. 타깃 칩**
어린이, 10대, 20대, 30대, 전연령 등 타깃 필터.

**4. 필터 초기화**
선택된 필터 모두 해제.

**5. 적용 버튼**
필터 적용 후 결과 갱신.

---

**3-1-2 탐색_상세 필터**

**1. 협업 유형 필터**
라이선싱, 콜라보 상품, 이벤트 등 가능 협업 유형 필터.

**2. 협업 범위 필터**
국내, 해외, 전세계 등 가용 범위.

**3. 예산 범위 슬라이더**
최소/최대 예산 범위 설정. 단위: 만원.

**4. 인증 상태 필터**
사업자 인증, Pro 멤버 등 인증 여부 필터.

**5. 응답률 필터**
응답률 80% 이상 등 필터 옵션.

---

**3-1-3 탐색_정렬**

**1. 기본 정렬 옵션**
최신순, 인기순, 조회순, 스크랩순. 모든 사용자 이용 가능.

**2. Pro 전용 정렬 (잠금)**
SNS 팔로워순, IP 신뢰지수순, 협업 활동지수순, 응답속도순 등. 클릭 시 요금제 업그레이드 안내 및 [9-4-2-요금제_BP] 링크.

---

**3-2 IP 상세**

**1. 메인 이미지 갤러리**
대표 이미지 및 추가 이미지 썸네일. 클릭 시 라이트박스 확대.

**2. IP 기본 정보**
IP명, 크리에이터명, 카테고리, 스타일 태그, 간략 소개.

**3. 스크랩/공유 버튼**

라이선싱, 콜라보 상품, 이벤트 등.

최소/최대 예산 슬라이더.

1주일 이내, 1개월 이내, 전체.

IPH만 활성화. 클릭 시 [8-1-프로젝트-지원] 플로우 시작. 마감된 프로젝트는 비활성화.

**5. 탭 네비게이션**
개요, 모집 분야, 디자인 방향, 예산, 혜택, 유의사항 탭.

캐릭터, 일러스트 등 원하는 IP 카테고리.

**2. 선호 스타일**
귀여운, 트렌디, 레트로 등 스타일 태그.

**3. 타깃 오디언스**
프로젝트 타깃 연령대/성별.

---

**4-2-3 프로젝트 상세_디자인 방향**

**4-2-4 프로젝트 상세_예산**

최소-최대 예산. 협의 가능 여부 표시.

선금/잔금 비율, 결제 시점 등.

## 10. Chat (채팅)

---

**10-1 채팅**

**1. 채팅방 목록**
진행 중인 채팅방 목록. 마지막 메시지 미리보기, 시간, 안읽은 개수.

**2. 채팅방 필터**
전체, 협업 진행 중, 완료.

**3. 채팅방 클릭**
[10-2-채팅방-상세]로 이동.

---

**10-2 채팅방 상세**

**1. 채팅 헤더**
상대방 프로필, IP/프로젝트명, 협업 상태 배지.

**2. 메시지 영역**
텍스트, 이미지, 파일 메시지. 시간 표시.

**3. 입력창**
텍스트 입력, 파일 첨부, 이미지 첨부 버튼.

**4. 협업 상태 카드**
제안 대기, 조건 협의 중, 계약 진행 중, 완료 등 상태 표시.

**5. 협업 액션 버튼**
상태에 따라 다름: 제안 수락, 결제하기, 협업 완료, 리뷰 작성 등.

---

**10-2-1 채팅방_제안_수락**

**1. 제안 내용 확인**
협업 유형, 예상 비용, 일정 등 제안 상세.

**2. 수락/거절 버튼**
수락 시 조건 협의 단계로 전환. 거절 시 사유 입력.

---

**10-2-2 채팅방_결제**

**1. 결제 정보**
협업 비용, 수수료, 총 결제액 표시.

**2. 결제 수단**
등록된 카드 또는 새 카드.

**3. 결제 버튼**
결제 완료 후 협업 진행 단계로 전환.

---

**10-2-3 채팅방_협업 완료**

**1. 완료 확인**
양측 모두 완료 확인 시 협업 종료.

**2. 정산**
IPH에게 정산 금액 지급. 수수료 제외 금액.

---

**10-2-4 채팅방_상호 평가**

**1. 별점 평가**
1-5점 별점.

**2. 항목별 평가**
응답 속도, 협업 태도, 결과물 만족도 등.

**3. 코멘트**
텍스트 리뷰 작성.

**4. 제출 버튼**
평가 완료. 상대방 프로필에 반영.

---

**10-3 협업 문의하기**

**1. IP 정보 카드**
문의 대상 IP 요약 정보. 응답률, 응답시간 표시.

**2. 협업 유형 선택**
라이선싱, 콜라보 상품, 이벤트/팝업, 기타 중 선택.

**3. 회사/브랜드명**
문의자 회사 정보. 필수.

**4. 문의 제목**
협업 제안 제목. 필수.

**5. 문의 내용**
상세 협업 제안 내용. 협업 목적, 예상 일정, 예산 범위 등.

**6. 첨부 파일**
기획서, 참고 이미지 등. 선택.

**7. 문의 보내기 버튼**
전송 후 채팅방 생성, [10-2-채팅방-상세]로 이동.
